Airbnb Qlik project

The goal of the dashboarding project is to create a dashboard in Qlik Sense that provides insight into Airbnb rentals over the past year in Antwerp, Brussels and Ghent. This was an individual assignement and we could decide who our target audience is.

The project was divided into two parts. The first part was comming up with a story to tell through your dashboard and creating the dashboard. I decided to focus on tourists to help them choose an airbnb rental in these three major cities. Once my story was chosen, the next step was the design of a solid underlying data model. To get this schema available in Qlik Sense I had to Extract-Transform-Load my data with the Qlik Scripting Language.

When the dashboard was finished the second part of the assignement began. The Goal of this second part is to create a small web application that tells the story behind the data. To complete this part I created a web application using HTML, CSS and Javascript to load my dashboard visualisations through web intigration that controls how Qlik Sense interacts with external web applications.

Key learnings:

  • Creating a data model in Qlik
  • Creating insights into data using Qlik
  • Integrate visualizations from Qlik Sense dashboard into a web applications
