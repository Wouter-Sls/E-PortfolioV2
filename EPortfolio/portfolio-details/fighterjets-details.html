Fighterjet image classification

This group project was made for my Big Data course at Thomas More Geel. The goal of this assignment was to create an image classification model using deep learning techniques seen in class. First we had to collect a dataset of images that could be categorized into five different classes. These classes had to be challenging enough for our model so no 'cat vs dog vs horse'-type of classifications.

We chose to collect a dataset of five different fighter jets:

  • F-1117 Nighthawk
  • F-16 Fighting Falcon
  • F-22 Raptor
  • F-4 Phantom
  • MiG-29 Fulcrum
To create a state of the art deep learning model we used techniques like fastai / Pytorch modelling and transfer learning. By building a webscraper using Selenium framework, we collected +-150 images of each category to train our model. When our model was trained we applied ROC and AUC metrics to check the quality of our classification model. As an extra we also compared our model to the performance of Google's TeachableMachine. As we expected our model was the better one. The last step of this project was to create an interactive web application using Streamlit, where users can test our model by uploading an image of a fighter jet. After uploading an image of a fighter jet the web application will tell you what type of fighter jet it is and how certain it is expressed in a percentage.

Key learnings:

  • Creating a deep learning image classification model
  • Use different techniques to check the quality of a model
  • Implementing our model into an interactive web application
