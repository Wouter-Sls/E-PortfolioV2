Detecting free parkingspots

On behalf of Elision, together with five other students we optimized the charging infrastructure at the Corda Campus. We developed a proof of concept to maximize the utilization of the existing electric vehicle charging stations. Using AI-powered cameras, users can check for available charging stations or join a waiting queue. Those in the queue can communicate via a chat function to swap positions, and when charging is complete, drivers receive a signal to make way for the next in line.

This group project was part of the Project 4.0 course at Thomas More Geel. The proof of concept application boasts key features, including a chat function for seamless communication, a user-friendly design, and notifications for available charging stations and fully charged cars. Developed using Next.js, the application offers a real-time visual overview of the parking lot, displaying numbered spots to indicate availability. To achieve real-time updates, a Raspberry Pi captures images every 10 seconds, sending them to a database.

As the sole AI student within the team, my role was pivotal. Leveraging a YOLOv8 model, I crafted a Python script to detect cars, facilitating the updating of parking spot statuses in the cloud database through an AWS Lambda function. This implementation significantly streamlined parking management processes, enhancing efficiency and effectiveness.
